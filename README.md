# Guias Definitivos: Recursos para Construção de Aplicações com IA

Repositório com guias práticos criados a partir de testes extensivos com diversas APIs e modelos de IA.

## Guias Disponíveis

### Kokoro TTS via OpenRouter
- Guia completo do modelo TTS Kokoro 82M
- 8 idiomas suportados incluindo Português Brasileiro
- 54 vozes pré-definidas
- Custo: $0.62/milhão de caracteres

### MiniMax Image Generation (image-01)
- API de geração de imagens com suporte a Text-to-Image e Image-to-Image
- 1-9 imagens por requisição
- 8 aspect ratios
- Custo: $0.0035 por imagem
